# 📘 Assignment: Class Enrollment CRUD

## 🎯 Objective

Build a Python program that manages class enrollment records using CRUD operations. In this assignment, you will practice storing data, updating collections, validating user choices, and organizing program logic into clear functions.

## 📝 Tasks

### 🛠️	Create the Enrollment Menu

#### Description
Create a menu-driven program that lets a user manage student enrollments for a class. The menu should allow the user to add, view, update, and delete enrollment records until they choose to exit the program.

#### Requirements
Completed program should:

- Display a text menu with options for Create, Read, Update, Delete, and Exit.
- Store enrollment records in a Python list or dictionary.
- Keep asking for user input until the user chooses to exit.
- Validate menu choices and handle invalid input without crashing.


### 🛠️	Manage Enrollment Records

#### Description
Implement the CRUD features for the enrollment system. Each enrollment record should include at least a student name, student ID, and class name.

#### Requirements
Completed program should:

- Add a new enrollment record with student name, student ID, and class name.
- Display all current enrollment records in a readable format.
- Update an existing enrollment record by searching with student ID.
- Delete an enrollment record by student ID.
- Show a clear message when a record is not found.
